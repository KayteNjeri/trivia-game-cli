**A CLI Trivia Game**

A simple Command Line Interface (CLI) trivia game created using Node.js.
The game provides an engaging user experience by allowing users to answer questions, within a given timeline and receive feedback.


**Features**

-Multiple choice trivia questions (A, B, C).

-Score tracking system.

-Has a countdown timer (10 minutes for the whole game time).

-Instant feedback (if answer is correct or incorrect with explanations).

-Colored terminal output using Chalk.


**How to run**

-Run the following script 'node bin/index.js' on terminal to open the Trivia Game.

-Select correct option (1 to continue or 2 to exit).

-Game starts with 1st question, multiple choices and asks for an answer.

-Moves to end game if there are no more questions.

-Final score is given.
